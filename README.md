# airbnb-clone

<p><h2>Set up guide</h2>

```git clone https://github.com/habeeb-an/airbnb-clone.git```
<p>to install yarn package manager</p>

``` npm install -g yarn```

then

``` yarn install```

<p>to run vite</p>

```cd client```

```yarn dev```

to connect api

```cd api```


```npm install npx --save-dev```  

```npx nodemon index.js```
